The Data Science Codex

A collection of code and resources to serve as a starting point for data science projects.

The purpose of this repository is to provide code and resources that can be quickly adapted and used for data science projects. An emphasis is placed on having clear understandable code that can be easily replicated. I make use of inbuilt datasets in R so that there is no need to download external datasets.

General

  • Resources - Websites and references that I find helpful for data science projects.
  • Creating Formatted Spreadsheets - How to create a custom formatted spreadsheet report with the openxlsx R package.
  • Using Python and R Together - How to use python and R code together in the same Jupyter notebook with the rpy2 python package.

Data Visualization

  • Visualization Cookbook - A wide variety of data visualizations demonstrated.
  • Geospatial Data Analysis - A work-in-progress collection of maps.

Modeling and Machine Learning

  • Modeling Fundamentals - A primer on logistic and linear regression modeling with the classic Titanic dataset.
  • Modeling Workflows - Streamlined Tidyverse modeling workflows with the gapminder dataset.
  • Clustering - Using the k-means alogrithm to cluster data.
  • Machine Learning with Caret - Using the Caret library to do machine learning.
  • Predicting Fuel Economy - Predicting the fuel economy of cars.

NLP

  • Hello NLP - Testing out some NLP techniques on Wall Street Journal sample data
  • State of the Union Analysis - Analysis of the language in historical State of the Union addresses
